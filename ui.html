Reconectar tokens

Busca valores sueltos (pegados desde Figma) en las capas y los reconecta a los tokens que ya tengas importados en este archivo, cuando el valor coincide exactamente.

Ámbito
Opciones
Desambiguar con Figma (opcional)

Si rellenas esto, cuando un valor coincida con varios tokens a la vez, el plugin lee las variables reales de ese archivo de Figma y elige el token con el mismo nombre, en vez de dejarlo sin resolver. Se guarda solo en este navegador (no se envía a ningún sitio salvo a la API de Figma). Déjalo vacío si prefieres decidir tú los casos ambiguos a mano.
