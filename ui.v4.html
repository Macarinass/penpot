Reconectar tokens

Busca valores sueltos (pegados desde Figma) en las capas y los reconecta a los tokens que ya tengas importados en este archivo, cuando el valor coincide exactamente.

Ámbito
Opciones
Desambiguar con el nombre real de Figma (opcional)

Cuando un valor coincide con varios tokens a la vez (p.ej. dos colores distintos que hoy son el mismo hex), no hay forma de adivinar cuál es el correcto solo mirando el valor — hace falta el nombre real que tenía en Figma. Para eso: copia el link del frame de Figma y pídeselo a Claude en el chat; te devuelve este JSON ya generado a partir de las variables reales de ese frame, listo para pegar aquí. Si lo dejas vacío, esos casos no se tocan y se listan al final para que los revises tú a mano.
